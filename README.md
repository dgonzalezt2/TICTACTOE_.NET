# HOW TO RUN THE PROGRAM
To run the TicTacToe game, follow these steps:

* Clone the Repository:
```bash 
git clone https://github.com/dgonzalezt2/TICTACTOE_.NET.git
```

* Run the program.
```bash
dotnet run
```
